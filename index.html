C++ Edinburgh

C++ Edinburgh is a meet-up for C++ developers in Edinburgh. Join us for some great talks and chance to chat and network with local C++ developers. Everybody's welcome, even if you're new to C++, so please feel free to come along to C++ Edinburgh!

Stay informed by following us on Twitter or Google+, or by joining the mailing list.
