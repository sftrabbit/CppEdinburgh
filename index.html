C++ Edinburgh

C++ Edinburgh is a new meet-up for C++ developers in Edinburgh, Scotland. Join local C++ developers for great talks and some socialising and networking. Everybody's welcome, whether you've never touched a C++ compiler before or you're a metaprogramming genius. If you're interested, please come along to C++ Edinburgh!

Stay informed by following us on Twitter or joining us on Google+.
